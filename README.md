# Apo2Mol: 3D Molecule Generation via Dynamic Pocket-Aware Diffusion Models

Apo2Mol is a diffusion-based model for **de novo** 3D molecule design from apo protein pockets, capturing pocket flexibility by jointly generating the ligand and its holo-like pocket conformation. 
For further details, please refer to our [Apo2Mol](https://arxiv.org/abs/2511.14559) paper.
